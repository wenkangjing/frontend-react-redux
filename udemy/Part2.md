### What is Redux?

![GUI Mockup](./redux_mockup.png)

![Data and View](./redux_diagram.png)

- Redux (data)
  + a state container
  + application level state
  + a collection of all the data to describe the app
  + **a single object**

- React (view)
  + a collection of views
  + user interaction
